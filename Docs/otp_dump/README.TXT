This example requires an attached Serial UART Port on PORT 0.  The serial port on the
computer should be setup as 115200 8N1.

A serial terminal program is required to monitor the printf output of the example.  

Tera Term is recommended for Windows.
GTK Term or Minicom is recommended for Linux or Mac.

When the example is run, contents of the user OTP memory section are dumped to the
serial port. The USN of the chip is also sent to the serial port.
